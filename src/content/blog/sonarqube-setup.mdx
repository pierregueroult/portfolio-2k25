# SonarQube: A Complete Guide to Deployment, Integration, and Usage

## Introduction

SonarQube is a powerful open-source platform for continuous code quality and security analysis. It provides static code analysis to detect bugs, vulnerabilities, and code smells across multiple languages.

### ğŸ”¹ Manual Installation

1. **Download SonarQube:**
   ```bash
   wget https://binaries.sonarsource.com/Distribution/sonarqube/sonarqube-10.3.zip
   unzip sonarqube-10.3.zip
   cd sonarqube-10.3
   ```
2. **Start the Server:**
   ```bash
   ./bin/linux-x86-64/sonar.sh start
   ```
3. **Access Web UI:**
   Open `http://localhost:9000/` and log in with default credentials:
   - Username: `admin`
   - Password: `admin`

Modify `sonar.properties` file:

```properties
sonar.jdbc.username=sonarqube
sonar.jdbc.password=your_password
sonar.jdbc.url=jdbc:postgresql://localhost:5432/sonarqube
```

Restart the SonarQube server for changes to take effect.

```bash
./bin/linux-x86-64/sonar.sh restart
```

### ğŸ”¹ Analyze a Project

1. Create a `sonar-project.properties` file:
   ```properties

   sonar.sources=src
   sonar.host.url=http://localhost:9000

   ```
2. Run the scanner:
